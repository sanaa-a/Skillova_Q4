# Skillova Assessment- Question 4

A simple website analytics dashboard created using Grafana Cloud as part of the Skillova Trainee Assessment — Category 4: Grafana Dashboards.

## Project Overview

This project takes one week of website analytics data stored in a CSV file and visualizes it using Grafana Cloud.

The dataset contains:

- Date
- Page Views
- Signups

The dashboard contains exactly two panels:

1. **Page Views Over Time** — A line chart showing the number of page views for each day.
2. **Total Signups** — A Stat panel showing the total number of signups.
